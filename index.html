PolicyKit-gnome provides an Authentication Agent for PolicyKit that integrates well with the GNOME desktop environment

See http://www.freedesktop.org/wiki/Software/PolicyKit for lots of documentation, mailing lists, etc. about PolicyKit.

See also the file HACKING for notes of interest to developers working on PolicyKit-gnome.

Report bugs against PolicyKit-gnome at the GNOME bugzilla

http://bugzilla.gnome.org/

against the policykit-gnome product.
